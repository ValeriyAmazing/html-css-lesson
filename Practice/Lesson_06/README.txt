📋 ЗАДАНИЕ: "Стильный плакат"

Создай красивый плакат или открытку с текстовыми эффектами!

═══════════════════════════════════════════════════════════════

✅ ТРЕБОВАНИЯ:

1. Подключи красивый Google Font
2. Сделай заголовок с тенью (text-shadow)
3. Используй text-transform для акцентов
4. Поэкспериментируй с letter-spacing
5. Добавь разные font-weight и font-style

═══════════════════════════════════════════════════════════════

💡 ПОДСКАЗКИ:

- font-family: 'Pacifico', cursive;
- text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
- text-transform: uppercase;
- letter-spacing: 5px;
- line-height: 1.5;

Неоновый эффект:
text-shadow: 
    0 0 5px #00ffff,
    0 0 10px #00ffff,
    0 0 20px #00ffff;
