📁 В ЭТОЙ ПАПКЕ:
   • index.html - твой файл для работы
   • solution/ - папка с примером решения (открой ПОСЛЕ выполнения!)

──────────────────────────────────────────────────────────────────
📝 ЗАДАНИЕ:
──────────────────────────────────────────────────────────────────

Создай страницу-визитку о себе, используя теги из урока.

Страница должна содержать:
   1. Главный заголовок (h1) с твоим именем
   2. Раздел "Обо мне" (h2 + абзац)
   3. Раздел "Мои увлечения" (h2 + абзац)
   4. Раздел "Мои мечты" (h2 + абзац)

──────────────────────────────────────────────────────────────────
🎯 ТРЕБОВАНИЯ:
──────────────────────────────────────────────────────────────────

✓ Использовать правильную структуру HTML (DOCTYPE, html, head, body)
✓ Использовать заголовки h1 и h2
✓ Использовать абзацы <p>
✓ Можно использовать <br> для переносов внутри абзаца
✓ Добавить хотя бы один комментарий в код

──────────────────────────────────────────────────────────────────
💡 ПОДСКАЗКИ:
──────────────────────────────────────────────────────────────────

Теги, которые тебе понадобятся:
   <h1>...</h1>     - главный заголовок
   <h2>...</h2>     - подзаголовок раздела
   <p>...</p>       - абзац текста
   <br>             - перенос строки
   <!-- ... -->     - комментарий

──────────────────────────────────────────────────────────────────
🏆 БОНУС (для продвинутых):
──────────────────────────────────────────────────────────────────

• Добавь ещё один раздел на свой выбор
• Попробуй использовать несколько <br> подряд
• Добавь комментарии-заметки для себя
