📋 ЗАДАНИЕ: "Карточки товаров"

Создай красивые карточки товаров с использованием Box Model!

═══════════════════════════════════════════════════════════════

✅ ТРЕБОВАНИЯ:

1. Задай ширину карточки (width)
2. Добавь внутренние отступы (padding)
3. Добавь внешние отступы между карточками (margin)
4. Сделай красивую рамку (border) и скругление (border-radius)
5. Добавь тень (box-shadow)
6. Центрируй карточки с margin: 0 auto

═══════════════════════════════════════════════════════════════

💡 ПОДСКАЗКИ:

* { box-sizing: border-box; }

.card {
    width: 260px;
    padding: 25px;
    margin: 20px auto;
    border: 2px solid #ddd;
    border-radius: 15px;
    box-shadow: 0 4px 15px rgba(0,0,0,0.1);
}
